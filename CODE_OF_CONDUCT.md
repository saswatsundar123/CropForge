# Code of Conduct

CropForge follows the Python Community Code of Conduct and expects all
contributors, maintainers, and community participants to help keep the project
welcoming, respectful, and research-friendly.

## Expected Behaviour

- Be respectful of different backgrounds, disciplines, and experience levels.
- Assume good intent and ask clarifying questions before escalating conflict.
- Give and receive feedback constructively, with attention to reproducibility.
- Keep discussion focused on the work: crop models, simulations, data, tests,
  documentation, and user outcomes.

## Unacceptable Behaviour

- Harassment, intimidation, discrimination, or sustained disruption.
- Personal attacks, insults, or demeaning language.
- Sharing private information without consent.
- Publishing intentionally misleading scientific or software claims.

## Scope

This code applies in all CropForge project spaces, including GitHub issues,
pull requests, discussions, documentation channels, release workflows, and
community support spaces.

## Reporting

If you experience or witness unacceptable behaviour, contact the project
maintainer through the repository's GitHub issue or security/contact channels.
Reports should include the relevant context and links where possible. The
maintainer will review reports in good faith and may remove comments, close
threads, block users, or take other moderation action needed to protect the
community.
